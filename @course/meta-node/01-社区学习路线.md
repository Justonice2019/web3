👋 准备好开启 Web3 之旅了吗？
如果你刚开始接触 Web3，或者一直对 DApp 开发感兴趣但不知道从哪下手，这份学习路线就是为你准备的。从最基础的 Web3 概念入门，到掌握 React 和 Next.js，再到接触钱包连接、合约交互，最后实战几个真实项目，一步步带你搭建起完整的 Web3 开发能力。

不管你是转型 Web3，还是零基础想系统入门，这份路线都尽量帮你避坑、少走弯路，把重点知识和推荐资源都整理好了。

👉 前端学习路线图👈

| 阶段     |   关键词    |   学习重点                                                       |
| 1-2周    | 基础认知 | 区块链/Web3/钱包/NFT等核心概念     |
| 3-6周   | 技术打底 | React、Next.js、Solidity理解                  |
| 7-10周  | 工具掌握 | 钱包连接、合约调用、状态管理           |
| 11-14周 | 实战落地 | 项目结构、交互实现、综合能力           |
🌐 第1-2周：Web3 扫盲阶段（理解基础概念）

目标：掌握区块链、钱包、代币、NFT、智能合约等 Web3 基础概念，为后续开发奠定认知基础。

🎯 学习任务：
1.👉 《以太坊公开课》
2.👉 WEB3扫盲

🌐 第3-6周：前端开发基础（React & Next.js）

目标：掌握前端核心技术栈 React + Next.js，为构建前端做好准备。

🎯 学习任务：
1.👉 React 学习路径（2-3周）
2.👉 Next.js 系统学习（3-4周）
3.👉 辅助理解 Solidity 合约结构

🌐 第7-10周：前端进阶（DApp 技术栈）

目标：掌握 DApp 前端常用库，具备构建 Web3 钱包交互组件的能力。

🎯 学习任务：
1.熟悉以下 DApp 技术栈工具:
wagmi：DApp 状态管理 + 钱包交互核心库
ethers.js：合约调用、钱包签名等核心操作
rainbowkit：钱包连接 UI 组件库
Viem：与 wagmi 配合使用的轻量调用库
WalletConnect：跨钱包连接协议
2.实战任务：
-使用 wagmi + rainbowkit 编写一个钱包连接组件
-实现连接钱包、显示地址、断开连接等基本功能
🌐 第11-14周：完整项目拆解与实战

目标：通过真实项目场景练习，熟悉 DApp 架构设计，独立完成项目功能。

🎯 学习任务：
1.Stake 项目
-拆解合约逻辑 + 前端交互逻辑
-实现质押、解质押、收益查看等功能

2.Pledge 借贷项目
-DApp 前端（借贷逻辑）
-管理后台（项目状态、参数配置等）

3.NFT 市场项目
-展示 NFT 项目卡片、购买流程
-签名交易、查看持仓、展示交易记录等功能
